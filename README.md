# Лабораторные работы по курсу "Наука о данных"

В этом репозитории собраны практические работы, выполненные в рамках курса. Суть посвящена сравнению различных статистических оценок и методов, их реализации и анализу результатов.

**Лабораторные работы 1-3:** Методы оценки неопределённости (наследственной погрешности).
    -   Метод Монте-Карло
    -   Метод Крейновича
    -   Метод линеаризации (с применением метода комплексного приращения)

Лабораторные работы 5-6: Сравнение робастных оценок центра обрабатываемой совокупности.
    -   середина размаха, среднее арифметическое, усеченное среднее, медиана, ско, mae, mad, статистика критерия Граббса 
    -   аналитический расчёт и метод Монте-Карло
    
Лабораторная работа 8: Исследование соответствия заданного и реального уровней значимости при проверке гипотезы о равенстве средних.
